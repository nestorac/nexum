- Cómic

# Esquema de ficheros
`acontecimientos.md` - Los sucesos relevantes que acontecen con sus fechas y lugares.
`bestiario.md` - Un compendio de monstruos que pululan en la ambientación.
`diagrama.drawio` - Un diagrama que condensa el lore entero.
`estructura.md` - Características del lore e instrucciones para la IA sobre cómo y qué construir.
`filosofia.md` - Conceptos filosóficos que se desarrollan.
`img/` - Imágenes e ilustraciones.
`lugares.md` - Localizaciones relevantes con sus descripciones.
`Nexum.md` - Un resumen de la historia del lore, que conviene mover hacia documentos más específicos, como `resumen.md`.
`personajes.md` - Un compendio de personajes humanos o con forma humana.
`profesiones.md` - Listado de profesiones en este lore con sus características.
`README.md` - Este documento.
`ambientacion.md` - Lista de características que busco en este lore, habría que mover su contenido a otras secciones más adecuadas.
`caracteristicas.md` - Carfacterísticas de mi lore, este contenido está duplicado de  `ambientación`.
`entes.md` - Organizaciones, empresas, asociaciones... relevantes.
`facciones.md` - Listado de las diferentes facciones que aparecen en el relato.
`LICENSE` - La licencia CC0 que se aplica a este lore.
`manga/guion.md` - El guión del manga basado en este lore.
`oneshots/preludio.md` - El preludio a esta historia, en formato oneshot de rol.
`Preludio.md` - Este documento tiene pinta de colisionar con el anterior.
`razas.md` - Las diferentes razas que aparecen en este lore.
`resumen.md` El lore resumido en lo importante, que puede servir como semilla para la IA.
